# Kaviya Sekar — Personal Portfolio

A fully hand-coded personal portfolio website built from scratch using HTML, CSS, and vanilla JavaScript. No frameworks, no templates, no libraries.

🔗 **Live Site:** [kaviyasekar.github.io](https://kaviyasekar.github.io)

---

## Built With

- **HTML5** — semantic structure and layout
- **CSS3** — custom properties (variables), flexbox, CSS grid, responsive media queries, keyframe animations
- **Vanilla JavaScript** — DOM manipulation, state management, dynamic content rendering

## Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation
- Animated hero section with fade-up transitions
- Dynamic sections: Experience, Projects, Publications, Skills
- Password-protected inline content editor (no backend needed)
- Contact section with direct mailto links
- Hosted on GitHub Pages — zero cost, zero server

## File Structure

```
kaviyasekar.github.io/
├── index.html       # Main HTML structure
├── style.css        # All styling — layout, typography, components, animations
├── script.js        # Editor logic, state management, dynamic rendering
└── README.md        # This file
```

## Sections

- **Hero** — name, role, bio, call-to-action buttons
- **About** — background, education, contact info
- **Experience** — work history with tags
- **Projects** — 7 projects with stats, descriptions, results
- **Publications** — 2 IEEE conference papers
- **Skills** — grouped by category
- **Contact** — email, location, LinkedIn

## How to Run Locally

No build step needed. Just open `index.html` in any browser.

```bash
git clone https://github.com/kaviyasekar/kaviyasekar.github.io.git
cd kaviyasekar.github.io
open index.html
```
